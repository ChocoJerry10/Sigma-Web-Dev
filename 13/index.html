Entities, code tag and more

Entities are used to display reserved characters in HTML. For example, the less than sign (<) is used to create a tag in HTML. If you want to display the less than sign, you can use the entity &lt;.

Here are some common entities:

  • &lt; - less than
  • &gt; - greater than
  • &quot; - double quote
  • &apos; - single quote
  • &amp; - ampersand

The code tag is used to display code in HTML. For example, <h1>Hello World</h1> will display

Hello World

.

The pre tag is used to display preformatted text in HTML. For example, the following code:

    <h1>Hello World</h1>
    <p>This is a paragraph.</p>

Will display:

Hello World

This is a paragraph.

The kbd tag is used to display keyboard input in HTML. For example, Ctrl + C will display Ctrl + C.

The samp tag is used to display sample output in HTML. For example, Hello World will display Hello World.

The var tag is used to display variables in HTML. For example, x will display x.

The abbr tag is used to display abbreviations in HTML. For example, HTML will display HTML.

The q tag is used to display short quotations in HTML. For example, Hello World will display "Hello World".
